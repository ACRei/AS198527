# 关于我们
Rei Academic Network([AS198527](https://bgp.he.net/AS198527))，目前由 [ReiAC](https://rei.ac) 运营，是一个主要面向教育科研的全球性实验网络。

该网络旨在提供一个稳定、安全、高效的网络环境，为网络技术实验提供一个良好的网络环境。
